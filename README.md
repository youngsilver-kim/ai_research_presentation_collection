This repository documents my learning journey in Artificial Intelligence, focusing on fundamental concepts, mathematical foundations, and presentation-based study materials.

As I do not have a strong background in mathematics, I place particular emphasis on understanding complex theories intuitively and organizing them in a clear and structured manner.

## Objectives

* Strengthen core knowledge in Artificial Intelligence
* Build mathematical foundations (Linear Algebra, Probability, Statistics)
* Organize learning through presentation materials
* Track continuous learning progress

## Structure

* **fundamentals/**: Mathematical foundations and core theories
* **machine-learning/**: Key machine learning concepts and methods
* **deep-learning/**: Neural networks and advanced architectures

Each topic includes presentation files along with brief notes summarizing key ideas.
