# Machine Learning

This directory contains study materials and presentation slides covering fundamental machine learning concepts, algorithms, and evaluation methods.

The goal is to build a solid understanding of how machine learning models learn from data, including both theoretical foundations and practical perspectives.

## Objectives

* Understand core machine learning algorithms and their assumptions
* Learn how models are trained, evaluated, and improved
* Develop intuition for selecting appropriate models for different problems
* Organize learning through structured presentation materials

## Topics Covered

* Supervised Learning (Regression, Classification)
* Unsupervised Learning (Clustering, Dimensionality Reduction)
* Model Evaluation and Validation
* Overfitting and Regularization
* Loss Functions and Optimization Basics
* Feature Engineering

## Structure

Each subfolder represents a specific topic and may include:

* Presentation slides (PDF / PPT)
* Summary notes (key ideas and intuition)
* Supporting examples if necessary

## Learning Approach

* Focus on understanding model behavior rather than memorizing formulas
* Compare different algorithms to identify strengths and limitations
* Emphasize intuition and real-world applicability
* Connect concepts to downstream deep learning models
